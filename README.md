# iFood Marketing Analytics Project

## Overview
This project analyzes customer behavior using the iFood marketing dataset to:
- Perform exploratory data analysis
- Develop customer segments
- Build a classification model for campaign response
- Generate business recommendations
- 
## Repository Contents
- Jupyter/Colab notebook (.ipynb)
- Exploratory data analysis
- Customer segmentation model
- Classification modeling
- Business recommendations

## Tools
Python, Pandas, Scikit-learn, Google Colab

## Key Findings
- Identified customer segments based on spending behavior
- Predicted likely campaign responders
- Developed data-driven targeting recommendations

## Data Source
Based on the public iFood Data Analyst Case, analysis independently conducted by Nicholson Hartowidjojo.
